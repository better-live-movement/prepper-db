# Prepper DB
This is a app to manage your food stock.

## dependecies
you need
- Ruby 2.4.3
- bundler have to be installed

## install
- clone this repo
- run
```
bundle install
```
to install missing dependecies

- run
```
rails db:migrate
rails db:seed
```
- to initialize the database

## usage
- run
```
rails s
```
to start the app
